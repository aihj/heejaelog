### 목적

### **“당신이 몰랐던 송도를 발견하게 합니다”**

- 그냥 ‘정보’를 주는 게 아니라,
    
    **“이런 곳이 있었어?” 라는 놀라움을 전합니다.**
    
- 송도에 숨어 있던 F&B 공간, 스토리, 분위기를
    
    **감도 높게 큐레이션하여 소개합니다.**

### **서서울 감도의 콘텐츠, 송도에 최적화하다**

- 뉴메거진, 푸딘코 같은 **감도 있는 콘텐츠 스타일을 로컬에 맞춰 재해석**
    
    → ‘송도 감성'에 맞는 색감, 구도, 글쓰기 톤으로
    
- 기존의 광고성 콘텐츠와는 **결이 다른 매거진 스타일 콘텐츠 제공**

### **사용자 중심의 정제된 콘텐츠 경험**

- 사진, 카피, 큐레이션 방식 모두 **사용자의 감각과 시선을 고려하여 제작**
    
    → 단순한 정보 나열이 아니라, *"지금 당장 가고 싶게 만드는 콘텐츠"*
    
- 협찬/광고가 아니라 **경험 기반의 신뢰할 수 있는 리뷰**가 중심

### Tech Stack

> CRA, javascript, react query, Tailwind, Java

## 주요 기능

1. 광고
   맛집,  축제/행사, 코스 추천, 키워드 별 추천 맛집제공 
2. 축제/행사
    행사 포스터(이미지), 이름, 타입(전시, 축제, 박람회), 일정, 운영시간
3. 소셜 로그인 및 글 작성, 조회, 수정 기능
