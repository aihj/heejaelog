### 목적

- 사용자가 다양한 투자 시나리오를 직접 만들어 실험할 수 있도록 함
- 포트폴리오 수익률, 리스크 등의 데이터를 시각화하여 직관적으로 비교
- 투자에 대한 이해와 흥미를 높이고, 실전 감각을 키울 수 있는 도구 제공

### Tech Stack

> React Native, Java

## 주요 기능
#### 1) 포트폴리오 구성기

- 사용자가 자산군(예/적금, 주식, 부동산, 금, 채권 등)과 비중을 자유롭게 설정
- 여러 개의 포트폴리오를 동시에 구성 가능

#### 2) 비교/분석

- 포트폴리오 간 수익곡선, 성과지표를 시각적으로 비교
- 어떤 포트폴리오가 가장 수익률이 좋은지 순위 시각화
- 표 및 그래프 기반 시각화 제공

- 저장 및 공유 (선택 사항)

- 포트폴리오를 저장하고 링크로 공유할 수 있음 (선택 기능)
